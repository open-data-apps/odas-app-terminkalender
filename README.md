# ODAS App Termin Kalender

Termin-Kalender-App für den Open Data App-Store (ODAP)

Die App zeigt Termine für die jeweiligen Kalender an.

Die App ist eine "ODAP App V1".

## Funktionen

- Anzeigen der Termine
- Vollbildansicht
- Suchfunktion
